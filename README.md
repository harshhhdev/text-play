# text-play

Simple Java scripts to generate fun text

# What's this? 

It might seeem confusing at first, here's what it does:

### Numifier

```bash
Enter your text to numify:

# input
Hello World!

# output 
=> H3110 W0r1d!
```

*this script is a clone of my [numifier](https://github.com/harshhhdev/numifier) in Python*

### Mock text

```bash
Enter your text to memify:

# input
Hello World!

# output 
=> HellO wORLd!
```

# Building :rocket:

```bash
# Clone this project
git clone https://github.com/harshhhdev/text-play.git

cd text-play

java mock.java
java numify.java
```
